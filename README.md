### Use
```
  $ mvn clean install
  $ mvn test
  $ mvn spring-boot:run
```
### Features
- Interactive periodic table of elements.
- Elemental experimentation.
- Real time chemical reaction.
- Create flash cards.
- Test your knowledge with a quiz.
